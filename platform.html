Module 03

The Sentinel

Edge-SideAlways OnCompliance-Grade

Enterprise AI requires systematic governance — not trust. The Sentinel is a mathematical monitoring layer that operates at the edge, before any data reaches the internet, enforcing policy on every prompt, retrieval, and tool execution.

🔒
Edge-Side PII Redaction
SSNs, credit cards, medical IDs, and custom PII patterns are masked on the local Dell Pro Max server before any API call is made. Zero leakage.
📉
Drift Detection
Automated nightly tests against a Golden Dataset catch when your model starts hallucinating or degrading. You get an alert before users notice.
🧑‍💼
Human-in-the-Loop Gates
Define triggers: any action tagged "irreversible", any transaction over $10k, any email to external domains — routes to a manager approval queue.
🌐
Full Lifecycle Checks
Unlike cloud guardrails that only check outputs, Sentinel checks: Input prompt → RAG retrieval quality → Tool call safety → Final response accuracy.

Module 05

MCP Gateway

MCP StandardPolicy Enforced

A unified, policy-enforced gateway that exposes all your enterprise systems as MCP-compatible tool endpoints. Every tool call is authenticated, authorized, and logged. No agent can touch a system it isn't explicitly permitted to use.

🔌
Standardized Tool Registry
Connect Salesforce, SAP, ServiceNow, databases, ERPs, REST APIs — all exposed as MCP-standard tool endpoints any agent can call.
🚦
Policy-Gated Execution
Role-based access control on tool execution. The HR agent can call HR tools only. The Finance agent can read but not write. IT defines the policy.
📝
Full Audit Trail
Every tool invocation is logged: which agent, which tool, which parameters, which user triggered it, and what was returned. Complete accountability.
⚡
Zero-Config MCP Servers
Pre-built MCP connectors for popular enterprise systems. Point at your Salesforce instance, authenticate once, start using immediately.
